Makealotl Music

Makealotl Music is a collection of edutainment web games. Come play with Delia as you make, match and explore musical concepts.

  • Answer questions to clean out the Opera House's note infestation in Note Nab

  • Note Nab contains three levels -- can you complete all three?

  • Test your memory in both note reading and pitch matching in Make A Match

  • Create new tunes or learn familiar songs on The Lilypiano

  • While all the games on Makealotl Music are single player, The Lilypiano can be used cooperatively with friends, with everyone using either their own device or sharing the same device.

  • Makealotl Music features fun pixel art

  • Each game contains high contrast UI and a dyslexic friendly font for maximum accessibility.
  • All Makealotl Music games are system agnostic -- that means they will all run on any internet conencted device, to include a tablet, chromebook or phone.

  • Grown ups will also enjoy the printables page for offline play.

  • For ages 5 and up. Some games may require reading skills.
